## 实验内容

在RK2206实训箱上使用ADC接口，实现MQ2烟雾传感器的编程开发。

## MQ2 API说明

ADC相关API详见PPT。

### 函数说明

#### (一)、MQ2 初始化

| 定义：       | mq2_init(void)                                                      |
| ------------ | ------------------------------------------------------------------- |
| **功能：**   | **初始化MQ2烟雾传感器模块**                                         |
| **参数：**   | **void**                                                            |
| **返回值：** | **void**                                                            |
| **依赖：**   | **\\code_base\\b6_adc_mq2\\include\\mq2.h**                         |
| **Tips:**    | **该接口只需调用一次，不支持重复调用。内部会初始化ADC通道6，用于读取传感器模拟电压值** |

#### (二)、MQ2 反初始化

| 定义：       | mq2_deinit(void)                                                    |
| ------------ | ------------------------------------------------------------------- |
| **功能：**   | **反初始化MQ2烟雾传感器模块，释放ADC资源**                          |
| **参数：**   | **void**                                                            |
| **返回值：** | **void**                                                            |
| **依赖：**   | **\\code_base\\b6_adc_mq2\\include\\mq2.h**                         |
| **Tips:**    | **释放ADC通道资源，应在不再使用传感器时调用**                      |

#### (三)、MQ2 校准

| 定义：       | mq2_ppm_calibration(void)                                           |
| ------------ | ------------------------------------------------------------------- |
| **功能：**   | **校准MQ2传感器，计算干净空气中的基准阻值R0**                       |
| **参数：**   | **void**                                                            |
| **返回值：** | **void**                                                            |
| **依赖：**   | **\\code_base\\b6_adc_mq2\\include\\mq2.h**                         |
| **Tips:**    | **应在干净空气中进行校准，校准环境PPM值默认为20。校准后R0值会保存在静态变量中** |

#### (四)、MQ2 读取烟雾浓度

| 定义：       | get_mq2_ppm(void)                                                   |
| ------------ | ------------------------------------------------------------------- |
| **功能：**   | **读取MQ2烟雾传感器的烟雾浓度值**                                   |
| **参数：**   | **void**                                                            |
| **返回值：** | **double，烟雾浓度值，单位：PPM**                                    |
| **依赖：**   | **\\code_base\\b6_adc_mq2\\include\\mq2.h**                         |
| **Tips:**    | **函数内部会读取ADC电压值，计算传感器阻值Rs，然后根据公式转换为PPM值。使用前需先进行校准** |

### 常量说明

#### ADC 通道

- `MQ2_ADC_CHANNEL`: 6 (对应GPIO0_PC6)

#### 校准参数

- `CAL_PPM`: 20 (校准环境中的PPM值)
- `RL`: 1 (负载电阻阻值，单位：kΩ)

```r
hardware_LIBS = -lhal_iothardware -lhardware -lshellcmd -lmq2_example
```

## 注意事项

1. **初始化顺序**: 必须先调用`mq2_init()`进行初始化，然后调用`mq2_ppm_calibration()`进行校准，最后才能读取数据
2. **ADC通信**: 传感器使用ADC通道6（GPIO0_PC6），ADC参考电压为3.3V，分辨率为10位（0-1023）
3. **校准环境**: 校准应在干净空气中进行，确保环境PPM值接近20，以获得准确的基准阻值R0
4. **数据单位**: `get_mq2_ppm()`返回烟雾浓度，单位为PPM（百万分之一）
5. **电压转换**: ADC原始值会自动转换为电压值（0-3.3V），然后根据传感器特性曲线计算PPM值
6. **预热时间**: 传感器首次上电后需要一定预热时间，建议等待30秒后再进行校准和读取

## 技术规格

- **通信接口**: ADC（模拟信号）
- **ADC通道**: 6 (GPIO0_PC6)
- **ADC分辨率**: 10位（0-1023）
- **参考电压**: 3.3V
- **输出**: 烟雾浓度（PPM）
- **工作电压**: 5V（传感器供电）
- **检测气体**: 液化气、丙烷、氢气、甲烷、酒精、烟雾等
- **响应时间**: < 10秒
- **预热时间**: 约20-30秒
