## 实验内容

在RK2206实训箱上使用PWM接口，实现SG90舵机的编程开发。

## SG90 API说明

PWM相关API详见PPT。

### 函数说明

#### (一)SG90 初始化

| 定义：       | sg90_init(void)                                             |
| ------------ | ------------------------------------------------------------ |
| **功能：**   | **初始化 SG90 舵机模块**                                    |
| **参数：**   | **void**                                                     |
| **返回值：** | **void**                                                     |
| **依赖：**   | **\\code_base\\b1_pwm_sg90\\include\\sg90.h**               |
| **Tips:**    | **该接口只需调用一次，不支持重复调用。内部会初始化 PWM5_M0 端口，频率为 50Hz** |

#### (二)SG90 反初始化

| 定义：       | sg90_deinit(void)                                            |
| ------------ | ------------------------------------------------------------ |
| **功能：**   | **反初始化 SG90 舵机模块**                                   |
| **参数：**   | **void**                                                     |
| **返回值：** | **void**                                                     |
| **依赖：**   | **\\code_base\\b1_pwm_sg90\\include\\sg90.h**               |
| **Tips:**    | **释放PWM资源，停止舵机控制**                                |

#### (三)SG90 设置角度

| 定义：       | sg90_set_angel(sg90_angel_e angel)                           |
| ------------ | ------------------------------------------------------------ |
| **功能：**   | **设置 SG90 舵机角度**                                       |
| **参数：**   | **`angel`: 角度枚举值(0°、45°、90°、135°、180°)**            |
| **返回值：** | **void**                                                     |
| **依赖：**   | **\\code_base\\b1_pwm_sg90\\include\\sg90.h**               |
| **Tips:**    | **舵机旋转角度范围：0-180度，频率为50Hz，占空比根据角度自动计算** |

### 常量说明

#### PWM 端口

- `EPWMDEV_PWM5_M0`

#### 角度枚举

- `SG90_ANGEL_0`: 0度
- `SG90_ANGEL_45`: 45度
- `SG90_ANGEL_90`: 90度
- `SG90_ANGEL_135`: 135度
- `SG90_ANGEL_180`: 180度

## 注意事项

1. **初始化顺序**: 必须先调用 `sg90_init()` 再进行角度设置
2. **PWM 通信**: 使用PWM5_M0端口，频率为50Hz
3. **角度范围**: 舵机角度范围为0-180度
4. **占空比计算**: 不同角度对应不同的占空比，驱动已自动计算
5. **电源要求**: 确保舵机有足够的电源供应

## 技术规格

- **通信接口**: PWM
- **PWM 端口**: PWM5_M0
- **频率**: 50Hz
- **角度范围**: 0-180度
- **占空比范围**: 0.5ms-2.5ms (对应0-180度)
- **工作电压**: 4.8V-6V
- **工作电流**: 100mA-250mA
